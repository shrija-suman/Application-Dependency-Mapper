# Application Dependency Mapper

## Overview

- Tool for visualizing application dependencies
- Built with Node.js backend and HTML frontend

## Features

- REST API for dependency data
- Simple web interface
- Docker support

## Architecture

- Backend: Express.js
- Frontend: Vanilla HTML/JS
- Data: JSON file
